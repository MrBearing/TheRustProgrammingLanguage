# 17章メモ

## 17.2トレイトオブジェクト〜〜

下記の実装はちょっと気になった。

```rust
pub struct Screen<T: Draw> { //<- Vectorに格納する型をScreen指定てのはちょっと。。
    pub components: Vec<T>,
}

impl<T> Screen<T>
    where T: Draw {
    pub fn run(&self) {
        for component in self.components.iter() {
            component.draw();
        }
    }
}
```

上の実装は悪例として紹介されていた。。この実装だと、Tの型が1種に限定されてしまい、多相性が損なわれる。

ダックタイピングについて述べていたが、厳密に言えばダックタイピングではない。
